# hits 

Self Hosted &bull; Minimal &bull; Hits Server

&rarr; Register Hits

```sh
curl https://domain.name/hit?url=https://another.domainname.com
```

&larr; Get Back Hits
```sh
curl https://domain.name/hits?url=https://another.domainname.com
```
